• accountInit

    • 初始化账户，成功则返回1，初始化账户时记录的医疗信息数初始化为0
    input:
    uint64 id
    string name
    string sex
    uint64 idCard
    int age
    uint64 tel

    output:
    int 1

  • accountUpdate

    • 根据账户的标识，id索引找到账户，这里设计的是只能更新年龄age和联系方式tel。
    input:
    uint64 id
    int age
    uint64 tel

    output:
    int 1

  • accountQuery

    • 查询账户信息，相比初始化时，增加返回医疗信息数量，建议在查询医疗信息时先通过此接口查询到医疗信息数量后再进行医疗信息查询的接口查询，避免越界。如有需要丰富查询，增加效率，建议把每个账户的医疗信息列表同步到数据库中优化。 input:
    uint64 id

    output:
    uint64 id
    ​ string name
    ​ string sex
    ​ uint64 idCard
    ​ int age
    ​ uint64 tel
    uint64 times

  • medicalRecord

    • 记录一条医疗信息，输入账户id和医疗信息，记录时账户对应的数据结构中times字段+1，医疗信息记录在账户数据结构中的医疗信息列表中。 input:
    uint64 id,
    uint64 time,
    string place,
    string info,
    uint64 timeStamp

    output:
    int 1

  • meidcalQuery

    • 查询某个用户的某一条医疗信息。输入为用户标识、第几条医疗信息，返回该条医疗信息。 input:
    ​ uint64 id.
    ​ uint64 times

    output:
    ​ uint64 time,
    ​ string place,
    ​ string info,
    ​ uint64 timeStamp
